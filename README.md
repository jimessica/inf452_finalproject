## Overview
NeuroFlow is an interactive desktop application designed to assist users, particularly neurodivergent individuals such as those with ADHD or autism spectrum conditions, in managing focus, stress, and cognitive load during daily tasks. The application provides a user-friendly interface that allows users to customize their work or study sessions, helping them enhance productivity and well-being.


## Features

### Customizable Focus Sessions
1. Stress and focus state: Users can input their current stress and focus state to get custom suggested session settings to match their needs, reducing decision fatigue.
2. Setting variables: A variety of settings are available to improve focus and calmness: time length, font, colour palette
3. Setting customization: Users still have the freedom to customize the session to their liking.

### User Interface
1. Minimalistic Design: The application features a simple and intuitive graphical user interface that minimizes distractions.
2. Accessibility Options: NeuroFlow emphasizes inclusivity with:
  - Ability to change font type, size, and color contrast at any time
  - Automatic break timers after the time is up
  - High color contrast the passes WCAG AAAA

### Focus Session Plan
1. The application generates a customized focus session plan by setting timer length, font, and colour palette. 


## Installation
- System Requirements: Ensure your system meets the following requirements:
- Operating System: Windows, macOS, or Linux
- Minimum RAM: 4 GB
- Minimum Disk Space: 100 MB
- Download: Obtain the latest version of NeuroFlow from the official website or repository.
- Installation Steps:
    1. For Windows: Run the installer and follow the on-screen instructions.
    2. For macOS: Drag the NeuroFlow application to your Applications folder.
    3. For Linux: Extract the downloaded package and run the executable.


## Usage
- Launching the Application: Open NeuroFlow from your applications menu or desktop shortcut.
- Setting up a session
  1. Select the desired session length.
  2. Input your current emotional or attentional state.
  3. Choose from the available sensory regulating tools.
- Starting the Session: Click the "Start Session" button to begin. Follow the audio and visual cues provided throughout the session.
